# MeuBlog

Blog estatico criado com Jekyll, publicado no GitHub Pages.

## Como publicar

Ver o ficheiro `tutorial-github.docx` incluido na pasta.

## Adicionar artigo

Cria um ficheiro em `_posts/` com o formato `AAAA-MM-DD-titulo.md` e faz git push.

## Desenvolvimento local

```bash
bundle install
bundle exec jekyll serve
```

Abre http://localhost:4000
